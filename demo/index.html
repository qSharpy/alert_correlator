What the App Does

The Alert Correlator is designed to solve a common problem in modern monitoring systems: alert fatigue.

When a system issue occurs, it often triggers multiple related alerts, overwhelming operators with notifications.

This application:

  • Receives alerts from Prometheus AlertManager
  • Groups related alerts into sessions
  • Uses AI to analyze alert patterns and determine root causes
  • Generates comprehensive incident reports
  • Provides runbooks for common scenarios
  • Visualizes correlation data in Grafana

Alert Correlation Flow

Application Overview

Detailed Alert Correlation Flow

The system integrates multiple components to create a complete alert correlation pipeline:

  1. External Systems send alerts to AlertManager
  2. AlertManager forwards alerts to the Backend Service
  3. Backend Service processes alerts and logs to Loki
  4. LLM Service analyzes alerts with AI and accesses runbook knowledge
  5. Grafana visualizes the correlation results and logs
